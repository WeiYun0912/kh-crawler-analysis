Crawler Noise Inspector

把這個書籤加進瀏覽器，逛任何網頁時點一下，右上角立刻浮出視窗顯示這頁有什麼 noise pattern。 不用裝 extension、不用下載檔案再丟到分析器 — 直接在瀏覽網站時就知道結果。

↓ 把下面按鈕拖到瀏覽器書籤列 ↓
⚓ Noise Inspector
或：右鍵 → 加到我的最愛 / 加入書籤
或：複製按鈕的連結（右鍵 → 複製連結）→ 手動建立書籤
§ 01 · 安裝步驟

怎麼裝

  1. 確認瀏覽器顯示書籤列（Chrome: Ctrl + Shift + B 切換）
  2. 把上面的「⚓ Noise Inspector」按鈕用滑鼠拖到書籤列
  3. 完成！
§ 02 · 怎麼用

怎麼用

  1. 逛任何網站，例如 https://coia.kcg.gov.tw/web_tw/index.php
  2. 點書籤列上的「⚓ Noise Inspector」
  3. 右上角浮出檢測結果，列出偵測到的 noise pattern（T01 ~ T10）
  4. 需要的話按「下載 .json」→ 拿到一個檔，拖進 crawler-document-analyzer.html 做深度分析
  5. 或按「複製為 JSON」直接放剪貼簿
§ 03 · 兩個版本

選一個版本裝

Loader 版 推薦

從 GitHub 經由 jsdelivr CDN 載入。規則更新會自動同步（commit 到 main → 隔天所有用戶自動拿到新版）。


    

    

      
Inline 版 離線可用

      
整段程式碼塞進 URL。不需要外部依賴，離線/內網也能用。但更新規則時要手動換書籤。

      
URL 太長無法直接顯示，請用上方按鈕拖曳，或下方「複製 Inline 版 URL」。
